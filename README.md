# じゃんけんゲーム - 文法の確認
グー 1, チョキ 2, パー 3をコンソールから入力し、
乱数で発生した手と対戦して結果を出力する。

## 設計
出した手と相手の手の関係は以下になる。
結果は、自分と相手の手の差分を3で割った余りで決まる。

|  あなた  |  あいて  | 差分を3で割った余り | 結果 |
| ---- | ---- | ---- | ---- |
|  1  |  1  | 0 | 引き分け |
|  1  |  2  | 2 | 勝ち |
|  1  |  3  | 1 | 負け |
|  2  |  1  | 1 | 負け |
|  2  |  2  | 0 | 引き分け |
|  2  |  3  | 2 | 勝ち |
|  3  |  1  | 2 | 勝ち |
|  3  |  2  | 1 | 負け |
|  3  |  2  | 0 | 引き分け |
